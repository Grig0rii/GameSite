# GameSite

Веб-сайт для загрузки и управления играми, построенный на Django.

## Описание

GameSite - это платформа, которая позволяет пользователям загружать, просматривать и скачивать игры. Проект включает в себя:

- Систему регистрации пользователей
- Загрузку игр с файлами
- Просмотр списка игр
- Детальную информацию об играх
- Административную панель

## Технологии

- **Backend**: Django 4.2.7
- **Frontend**: HTML, CSS, JavaScript
- **База данных**: SQLite
- **Медиа файлы**: Pillow для обработки изображений

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/GameSite.git
cd GameSite
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Выполните миграции:
```bash
python manage.py migrate
```

5. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

6. Запустите сервер разработки:
```bash
python manage.py runserver
```

## Структура проекта

```
GameSite/
├── backend/          # Настройки Django
├── games/            # Приложение для игр
├── media/            # Медиа файлы (игры, изображения)
├── templates/        # HTML шаблоны
├── manage.py         # Управляющий скрипт Django
└── requirements.txt  # Зависимости Python
```

## Использование

1. Откройте браузер и перейдите по адресу `http://127.0.0.1:8000/`
2. Зарегистрируйтесь или войдите в систему
3. Загрузите игру через форму загрузки
4. Просматривайте доступные игры в списке
5. Скачивайте игры, нажав на кнопку скачивания

## Администрирование

Для доступа к административной панели:
1. Перейдите по адресу `http://127.0.0.1:8000/admin/`
2. Войдите с учетными данными суперпользователя
3. Управляйте играми и пользователями
